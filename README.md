### Particle System
 * Particle System
####  1 - Dependencies

* MySql
* Webpack
* Three.js

 #### 2 - Run Data Base
 In the Terminal execute command:
...

  cd src
...
  node app.js
 ####  3 - Run project
In another Terminal execute command to resolve dependencies:
```
npm i
```
To create the server:
```
npm run webpack-dev
```
** Open in the browser http://127.0.0.1:5500/src/client/template.html **
