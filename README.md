### DOMEVIDEOPLAYER
 * DOMEVIDEOPLAYER [Demo](). 
####  1 - Dependencies


* Webpack
* Babel
* Three.js
* GLTF Models Loader

 ####  2 - Run project
In the Terminal execute command to resolve dependencies:
```
npm i
```
To create the server:
```
npm run webpack-dev
```
** Open in the browser http://127.0.0.1:5500/src/client/template.html **
